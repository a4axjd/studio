# **App Name**: SimpleHello

## Core Features:

- Text Display: Display a greeting text (HELLO) prominently on the landing page.
- Contact Link: Include a 'contact me' link, with the option to supply my own details to include as a `mailto:` URL.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey a sense of sophistication and modernity.
- Background color: Light gray (#FAFAFA) to provide a clean and neutral backdrop.
- Accent color: Lavender (#D8B4FE) to highlight the 'contact me' button.
- Body and headline font: 'Inter' sans-serif for a clean and modern aesthetic.
- Center the text (HELLO) horizontally and vertically on the page for maximum impact.
